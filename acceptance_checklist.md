# Acceptance Checklist

- [ ] Endpoints for list/details respond without errors.
- [ ] In case of API outage, the UI displays cached data or a user-friendly message.
- [ ] JSON-LD is valid and visible in the Page Source.
- [ ] TTFB for a detail page with a warm cache is less than 800ms.
- [ ] No sensitive keys are present in the frontend code.
- [ ] If an API key is required, it is used only through the optional proxy.
